# 💸 SplitBuddy — Smart Expense Sharing, Made Easy

SplitBuddy is a full-stack web app that helps you manage shared expenses without the usual confusion. Whether you're splitting bills with friends, roommates, or teammates, it keeps everything clear, fair, and easy to settle.

---

## ✨ What You Can Do

* 📱 **Use it anywhere**
  Clean, modern UI that works smoothly on both mobile and desktop.

* 🔐 **Quick login with Google**
  No sign-up hassle — just log in and start.

* 👥 **Add friends easily**
  Connect with people using their email and start sharing expenses.

* 💰 **Split expenses your way**
  Divide bills equally or customize who pays what, with live balance tracking.

* ⚡ **Smart “Settle Up” feature**
  Reduces the number of transactions so you don’t have to send multiple payments.

* 🔔 **Instant updates**
  Get notified when expenses are added or settled.

* 🛡️ **Secure & reliable**
  Includes rate limiting and safe API handling to prevent misuse.

---

## 🛠️ Tech Stack

**Frontend:** React, Tailwind CSS, Axios
**Backend:** Node.js, Express
**Database:** MongoDB
**Auth:** Google OAuth + JWT
**Other:** Redis, Nodemailer, Socket.io

---

## ⚙️ How It Works

### 📊 Expense Tracking

Add an expense, choose participants, and SplitBuddy automatically updates who owes whom in real time.

### 🔄 Smart Settlement Logic

Instead of multiple back-and-forth payments:

* Groups creditors (people who should receive money)
* Groups debtors (people who owe money)
* Matches them efficiently to reduce transactions

👉 Result: fewer payments, less confusion.
